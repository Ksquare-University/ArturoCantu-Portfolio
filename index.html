Custom Programming Language Project

This is a simple programming language designed and coded by me with the standard functions of an imperative programming language. My language supports the declaration of variables, functions, conditions, cycles, arrays as well as the declaration of a main function and some other features.

My language was built using Lex & Yacc for the language lexical and syntax analyzer while the functions, virtual memory emulation and virtualization were developed on Python. Memory is emulated using data structures and what I call a "virtual machine" executes the translated code which is formatted in quadruples and virtual memory addresses.
